Table of Contents

Namespace AICore.Services.Types.Data

Classes

AIServiceInitData

Represents the essential configuration required for initializing an AI service. This information includes the type of AI service, the API key, and the default AI model to be used.

VoiceServiceInitData

Represents the required configuration data to initialize a voice service. This configuration includes API Key, geographic region, locale, and choice of voice.

Enums

AIModel

Enum representing all the various AI models currently supported by the AICore. Each member of this enum corresponds to a particular AI model.

AIRole

Defines the roles associated with communication between the user and the GPT models. Each role reflects a type of interaction or source of message within the system.

AIServiceType

Defines the type of AI service provider that is utilized for AI-related operations. This may be expanded in the future to include other service providers.
