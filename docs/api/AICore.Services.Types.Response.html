Table of Contents

Namespace AICore.Services.Types.Response

Classes

AIEmbeddingResponse

Represents the response from the AI Embedding model. The response includes an array of data objects containing embeddings and corresponding indices, and also the model used for generating embeddings.

AIEmbeddingResponseData

Contains the details of each embedding generated by the AI embedding service. This includes the actual embedding vector and its corresponding index.

AIImageResponse

Represents the response from the AI Image service. This response includes timestamps and a list of successfully processed ImageUrls.

AITransformerResponse

Represents the response from generative pre-trained transformers models. It includes information on usage and generated data.

DataContent

Represents a single message of the conversation generated by the AI model. The message can include a role (either "system", "user", or "assistant"), content, and possible function calls.

FunctionContent

Represents a function call made during the AI model's conversation. The function call includes the name of the function, and the arguments.

GeneratedData

Represents a single generated message by the AI model's response. The message includes the role of the conversation participant, the content, and function call data (if a function was called).

ImageUrl

Represents a single image as a URL. This is used in the list of processed images within the AIImageResponse.

UsageData

Detailed information about the usage of tokens in the AI model's response.
