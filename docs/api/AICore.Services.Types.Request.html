Table of Contents

Namespace AICore.Services.Types.Request

Classes

AIEmbeddingRequest

Holds the necessary data required to send an embedding request to an AI model. It comes with a default choice of AI model.

AIImageRequest

Represents a request for generating AI images based on a given prompt. Users can specify the number of images and the size of the output images.

AITransformerRequest

Represents a request for the generative pre-trained transformers model. This request includes the sequence of messages, list of possible functions to call, and parameters such as model, tokens, temperature, and penalties.

DataMessage

Represents a single message in the conversation. This can be a message from the system, user, or assistant. It may also include a function call.

Function

A function that can be called during the conversation, its description, and details about its parameters.

FunctionCall

Details of a function call including its name and arguments.

FunctionDetail
FunctionParameter

Documents the parameters a function accepts, including their type and required status.
