Class NoSpoonAIClient

Namespace
AICore.Infrastructure.NoSpoonAI
Assembly
ai_core.dll

This class is a wrapper for IAIGPTService calls. It is a framework that allows interaction with IAIGPTService calls with the added benefit that we can build prompt objects. This allows us a high level of customization in handling prompts.
